
## 👨‍💻 About This Project

I built this project to practice **Computer Vision using OpenCV and MediaPipe**.
It helped me understand how hand tracking and gesture detection works in real-time.

# ✋ Finger Counter using OpenCV


This project uses **Computer Vision** to detect and count the number of fingers shown in front of a webcam.

## 🚀 Features

* Real-time hand detection
* Finger counting logic
* Works using webcam
* Beginner-friendly project

## 🛠️ Tech Used

* Python
* OpenCV
* MediaPipe

## ▶️ How to Run

1. Clone the repository:

   ```bash
   git clone https://github.com/sumit-singh-coder/finger-counter.git
   ```

2. Install dependencies:

   ```bash
   pip install -r requirements.txt
   ```

3. Run the project:

   ```bash
   python main.py
   ```

## 📸 Output

Shows number of fingers and fist detected in real-time.
